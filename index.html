Our Products

What We Provide

Sensors
Cameras

Dashboards

  • Sensor Module

    A modular and plug & play sensor system that can be attached to any machine to collect data and send it to the cloud. It is also an edge device that can be used to run machine learning models on the edge to detect anomalies.

    Temperature Pressure Vibration Current Plug and Play Anomaly Detection

  • Camera Module

    A modular and plug & play camera system that can be attached to any machine or production line to monitor the process and machine. It can be used for application like Counting poka yoke, defect detection, etc.

    Counting Poka Yoke Defect Detection Plug and Play Anomaly Detection

  • Gateways

    A centralised device that collects data from all the sensors and cameras and sends it to the cloud. It uses Mqtt protocol to send and receive data.

    Mqtt Protocol Plug and Play M2M Communication

  • HMI

    HMIs for associates to monitor and provide neccessary inputs. It can be used to display the data collected from the sensors and cameras.

    Data Visualization Digital inputs Alerts and Notifications

  • Dashboard

    A dashboard to monitor the machine metrics and process. We provide a customized dashboard for associates and admins.

    Condition Monitoring Predictive Maintenance Alerts and Notifications Data Visualization
